# وب‌سایت برند شخصی — مرضیه صدیقی

سایت برند شخصی مرضیه صدیقی، بنیان‌گذار کوشش گران قرن و معمار اتوماسیون و ایجنت‌های هوشمند.

## ساختار پوشه

```
marzieh-website/
├── index.html          صفحهٔ اصلی فارسی (RTL) — صفحهٔ پیش‌فرض
├── index-en.html       صفحهٔ اصلی انگلیسی (LTR)
├── assets/
│   ├── css/styles.css  استایل مشترک هر دو زبان
│   └── js/main.js      اسکریپت‌ها (نمودار ایجنت، اسکرول، هاور)
└── README.md           همین فایل
```

## نحوهٔ مشاهده

فقط کافی است `index.html` را با مرورگر باز کنی. هیچ نصب یا سروری لازم نیست.
دکمهٔ زبان در نوار بالا (EN ⇄ فا) بین نسخهٔ فارسی و انگلیسی جابه‌جا می‌شود.

## هویت بصری

- پس‌زمینه: شب‌رنگ زمردی عمیق (#07120D)
- رنگ اصلی: سبز زمردی (#37D08A / #1E7A4E)
- رنگ مکمل: کهربایی گرم (#E8B468) — فقط در نقاط تأکید
- فونت: Vazirmatn (از CDN بارگذاری می‌شود)

## نکات

- این نسخه، صفحهٔ اصلی (Landing) است. دکمه‌های «ارسال درخواست» فعلاً نمونه‌اند.
- برای نسخهٔ کامل (فرم درخواست، داشبورد کاربر و ادمین، رزرو و پرداخت)،
  این صفحه به پروژهٔ Next.js + Supabase تبدیل می‌شود.
- فونت Vazirmatn از CDN می‌آید؛ برای حالت آفلاین می‌توان فونت را
  به صورت محلی در assets قرار داد.
